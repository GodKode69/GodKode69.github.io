README.md

Monteria is a disocrd selfbot for the discord bot @Pokétwo#8236 which uses pokehint package to guess the wild pokemon and capture it using discord.js-selfbot-v13

The core focuses on two modes -
1. Spam mode - spam messages to spawn pokemons to catch
2. Incense mode - auto buy and use incense to catch pokemon

The bot also has utility commands, allowing you to use the bot with a prefix insetead of mentions.

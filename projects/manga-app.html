MANGA-APP

Native Desktop Reading Environment

Electron Node.JS MongoDB Cheerio.js React
README.md

A dedicated desktop reader built to solve the limitations of web-based manga aggregators. By utilizing the Electron framework, the app provides a native experience with no ads, hardware-accelerated rendering, and offline capabilities.

It features a custom scraper engine that unifies sources from multiple providers into a single, clean UI.
